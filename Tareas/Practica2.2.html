102 Etiquetas HTML

Inicio

Principales


1. html

Descripción: Define la raíz del documento HTML.

Ejemplo: <html>...</html>

Adicional: Contiene todo el contenido de la página.

2. head

Descripción: Contiene metadatos e información del documento.

Ejemplo: <head>...</head>

Adicional: No es visible para el usuario.

3. title

Descripción: Define el título que aparece en la pestaña.

Ejemplo: <title>Mi Página</title>

Adicional: Es obligatorio dentro de head.

4. body

Descripción: Contiene el contenido visible del documento.

Ejemplo: <body>Contenido</body>

Adicional: Solo puede haber uno.

5. h1

Descripción: Encabezado principal.

Ejemplo: <h1>Título</h1>

Adicional: Solo debe haber uno principal.

6. h2

Descripción: Encabezado secundario.

Ejemplo: <h2>Subtítulo</h2>

Adicional: Organiza jerarquía.

7. h3

Descripción: Encabezado nivel 3.

Ejemplo: <h3>Texto</h3>

Adicional: Uso estructural.

8. h4

Descripción: Encabezado nivel 4.

Ejemplo: <h4>Texto</h4>

9. h5

Descripción: Encabezado nivel 5.

Ejemplo: <h5>Texto</h5>

10. h6

Descripción: Encabezado nivel 6.

Ejemplo: <h6>Texto</h6>

11. p

Descripción: Define un párrafo.

Ejemplo: <p>Hola mundo</p>

12. br

Descripción: Salto de línea.

Ejemplo: Texto<br>Nuevo

13. hr

Descripción: Línea horizontal.

Ejemplo: <hr>

14. a

Descripción: Crea enlaces.

Ejemplo: <a href="https://google.com">Ir</a>

15. img

Descripción: Inserta imágenes.

Ejemplo: <img src="foto.jpg" alt="Foto">

16. ul

Descripción: Lista desordenada.

Ejemplo: <ul><li>Item</li></ul>

17. ol

Descripción: Lista ordenada.

Ejemplo: <ol><li>Item</li></ol>

18. li

Descripción: Elemento de lista.

Ejemplo: <li>Elemento</li>

19. div

Descripción: Contenedor en bloque.

Ejemplo: <div>Contenido</div>

20. span

Descripción: Contenedor en línea.

Ejemplo: <span>Texto</span>

Texto


21. strong

Descripción: Texto importante.

Ejemplo: <strong>Importante</strong>

22. em

Descripción: Texto enfatizado.

Ejemplo: <em>Énfasis</em>

23. b

Descripción: Texto en negrita.

Ejemplo: <b>Texto</b>

24. i

Descripción: Texto en cursiva.

Ejemplo: <i>Texto</i>

25. u

Descripción: Texto subrayado.

Ejemplo: <u>Texto</u>

26. small

Descripción: Texto pequeño.

Ejemplo: <small>Texto</small>

27. mark

Descripción: Texto resaltado.

Ejemplo: <mark>Texto</mark>

28. sub

Descripción: Subíndice.

Ejemplo: H<sub>2</sub>O

29. sup

Descripción: Superíndice.

Ejemplo: X<sup>2</sup>

30. code

Descripción: Código en línea.

Ejemplo: <code>print()</code>

31. pre

Descripción: Texto preformateado.

Ejemplo: <pre>Texto</pre>

32. blockquote

Descripción: Cita en bloque.

Ejemplo: <blockquote>Texto</blockquote>

33. q

Descripción: Cita corta.

Ejemplo: <q>Texto</q>

34. abbr

Descripción: Abreviatura.

Ejemplo: <abbr title="HyperText Markup Language">HTML</abbr>

35. address

Descripción: Información de contacto.

Ejemplo: <address>Correo</address>

36. cite

Descripción: Referencia a obra.

Ejemplo: <cite>Libro</cite>

37. bdo

Descripción: Dirección de texto.

Ejemplo: <bdo dir="rtl">Texto</bdo>

38. time

Descripción: Fecha u hora.

Ejemplo: <time>2026</time>

39. del

Descripción: Texto eliminado.

Ejemplo: <del>Texto</del>

40. ins

Descripción: Texto insertado.

Ejemplo: <ins>Texto</ins>
